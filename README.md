# DS-2026-1
Aplicação desktop em Java que converte texto em música MIDI. Cada caractere é mapeado para um evento musical (nota, pausa, troca de instrumento, etc.) e cada linha do texto se torna uma voz independente, permitindo composições polifônicas no estilo de fuga.

Projeto desenvolvido para a disciplina de Desenvolvimento de Software da UFRGS.
